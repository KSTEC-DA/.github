# 🏢 KSTEC Decision Automation 사업부
Decision Automation 사업부에서 개발하는 모든 프로젝트를 위한 레파지토리입니다.

## 조직 초대 및 권한


### 조직 초대
🚨 Github의 1인 1계정 원칙에 따라 기존 사용하시던 **개인 계정에 회사 이메일을 추가** 해야합니다.

1. GitHub 로그인 후 ["Settings > Emails"](https://github.com/settings/emails) 페이지로 이동합니다.
2. **Add email address** 항목에 회사 이메일을 추가합니다.
3. 회사 이메일로 발송된 인증 메일에서 **Verify**를 진행합니다.   
4. 회사 이메일 추가 후 관리자에게 조직 초대 요청을 합니다.


### 레파지토리 권한 요청
모든 레파지토리는 기본적으로 권한이 없으며, 필요한 경우 관리자에게 권한 요청을 해야합니다.
